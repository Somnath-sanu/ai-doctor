UI Redesign Rule:

When improving UI:
- Do NOT change business logic
- Do NOT modify API calls
- Do NOT change backend logic
- Only improve layout, components, and styling

Focus on:
- Modern SaaS UI (DARK UI WITH GRADIENT BG)
- Clean spacing
- Proper navigation (sidebar/navbar)
- Responsive layout

Design Style:
Modern SaaS dashboard style
Rounded corners
Soft shadows
Clean spacing
Minimalist layout
